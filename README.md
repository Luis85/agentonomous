# agentonomous

Autonomous agent library for TypeScript simulations. Engine-agnostic, fully
testable, and designed to nurture an agent from birth to death in the browser
with zero configuration.

**Status:** pre-release. Phase A milestones are tracked as issues in the repository.

## Quickstart (coming with M14)

## Development

```bash
nvm use               # node 22
npm install
npm test              # vitest
npm run typecheck     # tsc --noEmit
npm run lint          # eslint 9 flat config
npm run build         # vite library mode → dist/
```
